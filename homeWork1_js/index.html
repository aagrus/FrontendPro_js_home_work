Дз 1

  • Переменные.

    Попробуйте описать 10-15 своих характеристик через переменные. Учитывайте, что значение некоторых ваши характрестик можно будет изменить в будещем. Ваш код должен это поддерживать. Выведете результат в консоль. Выведите в консоль типы этих данных

  • Объект.

    Попробуйте описать себя через объект и вывести значение этого объекта в консоле. Все ваши харакретеристики можно выразить через различные типы данных: возврат - число, имя - строка, сутдент ли школы Hillel - через булеановское значение. Пример: const anton = { age: 31, name: 'Anton', isMaried: true, };

  • Модальные окна.

    Используя изученные модальные окна, составьте опросник (5-10 вопросов) пользователя. Тема опросника любая: Кино, музыка, спорт и т.д. Результат вывести в консоль.
